FAQ

Q1.How to create an account?

ANS.Register through the login page.


Q2.How to search for book?



ANS.The default search widget of the portal allows you to search for any book that is currently owned by the library.


Q3.How to view books of a particular type?



ANS.The search widget allow you to view the books of a particular type.


Q4.How to find the location of a particular book in the library?



ANS.The search widget allow you to view the books of a particular type.


Q5.How to request for a new book?



ANS.There's an option on your home page to request for a new book which requires the user to fill a form.


Q6.How to view the status of a book?



ANS.The search widget provides this feature also.


Q7.How to find out the books I have issued and their respective due dates?



ANS.There is a query given to the user which shows the books the user has issued and their respective due dates.


Q8.Is there anyway so that I can select my favourite topics?



ANS.At the time of creating an account you will be given choices of various topics from which you can select your favourites.There is a query which serves this purpose.Your account will show the updates on these selected topics.


Q9.How to see the list of people who have made reservation on the same book as I have?



ANS.The search widget shows the reservations on the book searched by the user.


Q10.What is the duration for which I can keep a book for a single issue?



ANS.You can keep the book for 14 days from the date of issue.
