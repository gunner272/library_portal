This project was done in a team of 2 as part of open-source development initiative taken for second semester
undergraduate program in IIIT-Hyderabad.The aim was to help in development of software aiding the IIIT-H community.
This virtual library portal acts as web interface ,where users can reserve,request books.
The librarian acts as the adminstrator of community who can comment ,rate books.
The librarian is given acces to interface which allows for controling logistics of library, with special automated system 
looking over late return of issued books.
